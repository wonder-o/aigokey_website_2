常见问题

先把常见顾虑说清楚，再决定开哪一档。

我需要自己配置 API 吗？

不需要。AigoKey 不是让你自己接 API、自己算 token 的服务。你按月开通后，就按套餐额度使用。

我该选轻量版、标准版还是高级版？

轻量使用选轻量版；日常稳定用选标准版；如果你明确觉得 $100/天不够，再看高级版。

三档月卡有什么区别？

主要区别是每日额度和月总额度。轻量版 $50/天，标准版 $100/天，高级版 $200/天。

高级版是不是无限用？

不是。高级版是更高额度：每天 $200，月总额度 $6000。适合密集任务，但仍然有明确额度边界。

Codex Agent 主要能帮我做什么？

Codex Agent 更适合处理需要执行的任务，例如阅读项目、修改代码、生成 Demo、整理技术方案、排查错误和配合 GPT 输出说明文档。

Image 系列模型适合哪些场景？

Image 系列模型适合生成封面、商品图创意、运营海报、文章配图、产品说明图和界面素材方向，配合 GPT 可以一起完成从文案到视觉的内容生产。

适合多人共用吗？

可以先联系客服确认你的使用方式。多人协作、统一开通或更高频使用，都建议先沟通清楚再开通。
